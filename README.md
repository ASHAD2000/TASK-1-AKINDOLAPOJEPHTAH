# DATALAB Internship – Project 1

## Project Title
Data Cleaning and Preparation using Microsoft Excel

## Objective

The objective of this project was to clean a raw e-commerce dataset by identifying and handling missing values, checking for duplicate records, and validating data formats to ensure the dataset was ready for analysis.

## Tools Used

- Microsoft Excel 2021

## Tasks Performed

- Identified missing values
- Replaced 308 missing CouponCode values with "No Coupon"
- Checked for duplicate records
- Validated date, number, currency, and text formats
- Prepared a cleaned dataset for analysis

## Files Included

- Raw Dataset
- Cleaned Dataset
- Data Cleaning Report
